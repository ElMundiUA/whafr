## What we now know about the corpus shape

The retriever is dominated by ≈ 12 narrow clusters:

- Agile / Scrum / SAFe / BDD / Cucumber (the heavy majority of hits)
- UI component libraries (Carbon, Ant Design, Polaris, TanStack)
- Playwright-cli MCP wrapper metadata
- Ant Design / Drizzle / Nx / Bun toolchain
- PostgreSQL install pages + Postgres.app
- Kubernetes service-mesh fragments (Istio multicluster)
- OpenTelemetry + Jaeger
- FedRAMP / NIST 199 (when "compliance" is asked)
- Pydantic + mypy name-drops
- Argo CD + GitHub Actions name-drops
- Refactoring Guru name-drops
- OWASP cheat-sheet *titles* (bodies missing — known ingest bug, fixed)

Everything outside this shape is empty. Concretely 100% gap on:
distributed-systems patterns, K8s runtime fundamentals, IaC,
real-time/CRDT, mobile platform code, network protocols, payments,
fintech compliance, performance engineering, language-specific
depth, RAG/MCP details (despite this being an MCP server), modern
PM frameworks, cloud provider services.
